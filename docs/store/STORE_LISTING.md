# Chrome 웹스토어 등록 정보

## 확장프로그램 이름
거북목 알리미 - Turtle Neck Reminder

## 간단한 설명 (132자 이내)
브라우저 사용 시간에 맞춰 자세 교정을 알려주는 확장프로그램입니다. 건강한 작업 습관을 유지하세요.

## 상세 설명
🐢 거북목 알리미는 오랜 시간 컴퓨터 앞에 앉아있는 분들을 위한 자세 교정 알리미입니다.

【주요 기능】
- ✅ 자동 타이머 - 브라우저를 켜면 자동으로 시간 측정 시작
- ✅ 시스템 알림 - 설정한 알림 시간(단일 기준)에 도달하면 자세 교정 알림 전송
- ✅ 2단계 상태 표시 - 시간 경과에 따라 `좋음 → 위험` 상태로 아이콘/문구 변경
- ✅ 시간 설정 - 알림 시간을 분 단위로 직접 설정 가능
- ✅ 알리미 선택 - 거북이/기린 캐릭터 중 선택 가능
- ✅ 다국어 지원 - 한국어/영어/일본어 지원
- ✅ 다크모드 - 시스템 설정에 맞춰 자동 적용

【사용 방법】
1. 설치 후 브라우저 툴바의 거북이 아이콘을 확인하세요
2. 아이콘을 클릭하면 현재 상태와 경과 시간을 볼 수 있어요
3. 스트레칭 후 "자세 교정 완료" 버튼을 눌러 타이머를 리셋하세요
4. 설정에서 알림 시간, 알림 on/off, 알리미 캐릭터를 조절할 수 있어요

【이런 분께 추천】
- 오랜 시간 컴퓨터 작업을 하는 개발자
- 재택근무로 자세가 무너지기 쉬운 분
- 거북목 예방에 관심 있는 분

건강한 자세로 건강한 하루 보내세요! 🐢

## 카테고리
웰빙

## 언어
한국어, 영어, 일본어

## 권한 설명
- alarms: 시간 측정 및 알림 예약
- storage: 설정 및 타이머 상태 저장
- notifications: 자세 교정 시간 알림 전송

## 개인정보 보호 (스토어 등록 양식용)

### 전용 목적 설명
```
이 확장프로그램은 사용자의 브라우저 사용 시간을 측정하여 일정 시간마다 자세 교정 알림을 보내는 건강 관리 도구입니다. 오랜 시간 컴퓨터 앞에 앉아있는 사용자가 거북목을 예방하고 바른 자세를 유지하도록 돕습니다.
```

### alarms 사용 근거
```
사용자가 설정한 알림 시간(예: 50분)에 도달했을 때 자세 교정 알림을 보내기 위해 타이머 기능이 필요합니다. chrome.alarms API를 사용하여 브라우저가 백그라운드에서도 정확한 시점에 알림을 전송할 수 있습니다.
```

### storage 사용 근거
```
사용자의 타이머 시작 시간과 설정(알림 시간, 알림 켜기/끄기, 알리미 캐릭터)을 로컬에 저장합니다. 이를 통해 브라우저를 껐다 켜도 설정이 유지되며, 사용자가 매번 다시 설정할 필요가 없습니다. 모든 데이터는 사용자의 로컬 저장소에만 저장되며 외부로 전송되지 않습니다.
```

### notifications 사용 근거
```
설정된 시간에 도달하면 사용자에게 자세 교정을 권유하는 시스템 알림을 보냅니다. 알림을 통해 사용자가 브라우저를 보고 있지 않아도 자세 교정 시간임을 인지할 수 있습니다. 알림은 사용자가 팝업에서 직접 끌 수 있습니다.
```
