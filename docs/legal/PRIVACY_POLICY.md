# 개인정보처리방침 (Privacy Policy)

**최종 수정일: 2025년 1월**

## 1. 개요

"거북목 알리미" 확장프로그램(이하 "본 확장프로그램")은 사용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다. 본 개인정보처리방침은 본 확장프로그램이 어떤 정보를 수집하고 사용하는지 설명합니다.

## 2. 수집하는 정보

본 확장프로그램은 **개인정보를 수집하지 않습니다.**

본 확장프로그램이 저장하는 데이터:
- 타이머 시작 시간
- 알림 설정 (첫 번째/두 번째 알림 시간)
- 알림 켜기/끄기 설정

이 데이터는 **사용자의 브라우저 로컬 저장소(chrome.storage.local)에만 저장**되며, 외부 서버로 전송되지 않습니다.

## 3. 정보 사용 목적

저장된 데이터는 오직 다음 목적으로만 사용됩니다:
- 브라우저 사용 시간 측정
- 사용자가 설정한 시간에 알림 전송
- 브라우저 재시작 시 설정 유지

## 4. 정보 공유

본 확장프로그램은 어떠한 데이터도 제3자와 공유하지 않습니다.

## 5. 데이터 보안

모든 데이터는 사용자의 로컬 브라우저에만 저장되며, 인터넷을 통해 전송되지 않습니다.

## 6. 사용자 권리

사용자는 언제든지:
- Chrome 확장프로그램 설정에서 본 확장프로그램을 삭제하여 모든 저장된 데이터를 제거할 수 있습니다.
- 팝업의 "자세 교정 완료" 버튼을 눌러 타이머를 초기화할 수 있습니다.
