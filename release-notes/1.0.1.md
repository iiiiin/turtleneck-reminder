# Release 1.0.1 (2026-01-27)

이번 업데이트는 알리미 캐릭터 선택 기능과 영어 지원을 추가한 기능 업데이트입니다.

## ✨ 주요 변경사항
- 🦒 새로운 알리미 캐릭터 '기린'이 추가되었습니다!
- 알리미 캐릭터를 고를 수 있는 셀렉트 박스를 추가했습니다.
- 영어(EN) 로케일을 추가해 다국어 사용성을 개선했습니다.
- 웹스토어 배포를 위해 `manifest.json` 버전을 `1.0.1`로 올렸습니다.

## 🔎 참고
- 웹스토어와 GitHub 버전 관리를 일치시키기 위해 태그는 `v1.0.1`을 권장합니다.
